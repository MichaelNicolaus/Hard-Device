Solusi AI

AI untuk lalu lintas, energi, layanan publik, dan banyak lagi.

Metodologi

a. Use Case Aplikasi

Platform ini bertujuan untuk mendeteksi kemacetan lalu lintas secara real-time di kota besar menggunakan data video dari CCTV dan sensor jalan, yang kemudian dianalisis oleh model AI untuk memberikan rekomendasi pengaturan lalu lintas.

b. Arsitektur Sistem

Sistem terdiri dari beberapa komponen utama:

  • Frontend: HTML, CSS (burn.css), dan JavaScript (burn.js).
  • Backend: Python Flask untuk API pengolahan data dan integrasi model AI.
  • Database: Firebase digunakan untuk menyimpan metadata hasil deteksi dan log aktivitas pengguna.

Tech stack: TensorFlow, OpenCV, Flask, Firebase, JavaScript.

c. Alur Dataset

Menggunakan dataset publik dari CityCam (video lalu lintas) dan dataset kendaraan dari Google Open Images. Dataset digunakan untuk melatih dan menguji model deteksi kendaraan berbasis YOLOv5.

d. Pengembangan Model

Model dikembangkan menggunakan YOLOv5 dengan framework PyTorch. Model dilatih menggunakan dataset kendaraan dan divalidasi dengan video lalu lintas untuk mengukur akurasi deteksi kemacetan.

e. Integrasi Model ke Environment Kode

Model AI dideploy sebagai REST API menggunakan Flask. Frontend mengakses model melalui request AJAX di JavaScript untuk mendapatkan hasil prediksi kemacetan secara real-time.

f. Integrasi dengan IoT Device

Pada tahap ini, IoT device tidak digunakan. Namun sistem dapat diperluas untuk integrasi ke sensor lalu lintas dan kamera berbasis Raspberry Pi atau ESP32 di masa depan.
